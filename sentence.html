Syarat & Rumus (The Rules)

Syarat mutlak terbentuknya sebuah kalimat adalah hanya memiliki satu Subjek (S) dan satu Prediket (P).

Komponen umum kalimat terdiri dari:

  • S = Subjek
  • P = Prediket
  • O = Objek
  • C = Complement (Pelengkap)

Rumus Valid (Benar) vs Tidak Valid (Salah)

Dalam menyusun kalimat, posisi penempatan harus tepat. Perhatikan rumus matematika bahasa berikut:

‚úÖ Pola Benar (Valid)

  • S + P + O + C
  • S + P + O
  • S + P
  • C + S + P + O
  • S + C + P + O
  • S + P + O + C + C

‚ùå Pola Salah (Invalid)

  • P + S (Prediket mendahului Subjek)
  • S + S + P (Double Subjek)
  • S + P + P (Double Prediket)

Kalimat Normal

Dikatakan "Kalimat Normal" apabila Subjeknya berasal dari keluarga Circle Noun (Kata Benda/Pronoun).

CONTOH ANALISIS

"I love you so much"

S (Subjek) P (Prediket) C (Complement)
I

(Circle Prediket)		 So much

Kalimat Tidak Normal

Dikatakan "Kalimat Tidak Normal" apabila Subjeknya BUKAN dari keluarga Noun, melainkan dari Circle Adverb.

CONTOH ANALISIS

"There is a book in my bag"

There
(Circle Adverb)		 Is a book
(Circle Prediket)		 In my bag
(Circle Adverb)
